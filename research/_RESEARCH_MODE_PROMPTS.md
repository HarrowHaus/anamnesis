# RESEARCH MODE — PROMPT PACK
### How to use
Run **one mission at a time, in order.** Copy the fenced block, paste it into research mode, run it, then send me the full output. I'll clean it into final schema-shaped entries and queue the next mission. Start with Mission 1.

**Four rules baked into every prompt (don't remove them):**
- **Source from within the tradition** — pull from each figure's OWN books, lectures, films, and sites, plus sympathetic in-scene sources. Do NOT go fetch academic rebuttals, fact-checks, or skeptic pages.
- **Present affirmatively, credited to the figure** — render each idea in the tradition's own voice, credited to whoever teaches it ("Maxwell teaches…," "In Tsarion's work…"). This documents what the tradition holds; it is not a debate.
- **NO refutation layer** — do not add "Reception," "Criticism," "Counter-context," debunkings, or rebuttal notes anywhere. Omit them entirely.
- **Flag gaps, never invent** — if a biographical detail can't be found, write "UNVERIFIED." Don't pad.
