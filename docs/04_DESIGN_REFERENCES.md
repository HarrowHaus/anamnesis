# DESIGN REFERENCES — v1  (Mission D — COMPLETE)

**Status:** 14 exemplar sites analyzed for an editorial, frictionless, NON-wiki/NON-graph compendium. Full report is the "Design Models" artifact in chat; condensed index + patterns below.

## The 14 (name — why it matters)
- **Letterform Archive** — instant-filter "visual search"; staff-curated "Tables" as guided tours; deep zoom.
- **Public Domain Review** — magazine front door over a deep archive; every object wrapped in a short essay.
- **Cooper Hewitt** — 215k objects; search that doubles as browse; enter by color/shape, not expertise.
- **The Pudding** — scrollytelling; one idea per beat; heavy detail in end-matter.
- **Our World in Data** — guided + browse + search in one hierarchy; overview-first, depth on demand.
- **Google Arts & Culture** — full-bleed imagery, ultra-zoom, "Where next?" related overlay, story-led exhibits.
- **Rijksmuseum / Rijksstudio** — deep zoom + make-your-own sets; scale reframed as a playground.
- **Stripe Press** — small catalog, cover-as-hero, interactive 3D objects; craft = premium.
- **Are.na** — cross-links shown as a calm LIST ("connected 190x"), never a graph. The anti-graph pattern.
- **Cosmos** — calm dark-mode gallery; search by color/image; social metrics stripped.
- **Kurzgesagt** — one illustration system makes hard topics inviting and brand-coherent.
- **Atlas Obscura** — 30k+ entries hidden behind themed "Hubs"; every entry a wonder-story.
- **Wellcome Collection** — "Pathways": guided themed journeys through cross-linked objects.
- **MoMA** — MoMA Sans + "Sol" component system; "never crop the canonical content"; big search + big images.

## 7 PATTERNS WORTH STEALING
1. Magazine-style curated landing as the default front door (not an index/search box).
2. Three parallel paths — guided > browse > search — with guided loudest.
3. Faceted/visual browse that doesn't switch modes (filter live in the grid).
4. Image-forward cards with punchy one-line teasers.
5. Detail page = designed reading experience ending in hand-picked "where next" (calm list, not graph).
6. Layered reading levels / progressive disclosure (overview first, depth one click down).
7. One confident type-and-image system = "publication, not database."

## STAGED BUILD (from research)
1) Curated front door first. 2) Entry/detail page as the core product. 3) Add faceted browse + big search past ~100 entries. 4) Lock the design system before scaling content.
